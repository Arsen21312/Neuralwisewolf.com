📚 Как умножать матрицы

Умножение матриц возможно только если количество столбцов матрицы A равно количеству строк матрицы B.

Если A = [m × n] и B = [n × p], то C = A × B = [m × p]

Каждый элемент результирующей матрицы C вычисляется по формуле:

C[i][j] = Σ (A[i][k] × B[k][j]) для k от 1 до n

Пошаговый процесс:

Шаг 1: Проверьте совместимость матриц (столбцы A = строки B)
Шаг 2: Для каждой ячейки результата перемножьте элементы строки A на элементы столбца B
Шаг 3: Суммируйте полученные произведения

📖Что такое умножение матриц?

Умножение матриц — это бинарная операция, которая создает новую матрицу на основе двух исходных. В отличие от умножения чисел, умножение матриц не коммутативно: A × B ≠ B × A.

Если матрица A имеет размер $m \times n$, а матрица B — $n \times p$, то их произведение C будет иметь размер $m \times p$. Каждый элемент матрицы C вычисляется как скалярное произведение соответствующей строки матрицы A и столбца матрицы B.

Пример вычисления элемента C[i][j]:

// Для матриц 2x2:
A = [a11, a12; a21, a22], B = [b11, b12; b21, b22]

C[1][1] = a11×b11 + a12×b21
C[1][2] = a11×b12 + a12×b22
C[2][1] = a21×b11 + a22×b21
C[2][2] = a21×b12 + a22×b22

Умножение матриц широко применяется в компьютерной графике, машинном обучении, решении систем линейных уравнений и многих других областях.
